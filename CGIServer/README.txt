Prima di avviare uno script dal WebServer, bisogna renderlo eseguibile.
Quindi:
	1 - Aprire un terminale
	2 - Andare nella cartella .../cgi-bin (contente gli script python da eseguire)
	3 - eseguire il comando "sudo chmod +x nomescript.py" - per fa si che il web server abbia i privilegi di esecuzione
